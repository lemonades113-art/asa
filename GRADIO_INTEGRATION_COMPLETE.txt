【使用方式】

1. 启动Web界面
   $ python agent_gradio.py

2. 打开浏览器访问（通常http://localhost:7860）

3. 选择Agent版本
   - 顶部"选择Agent版本"单选框
   - v1.0: 快速对话、轻量级任务（默认）
   - v2.0: 复杂分析、深度报告（可选）

4. 创建新会话
   - 输入用户名
   - 点击"初始化/新建会话"

5. 开始对话
   - 输入您的问题
   - 观察系统响应和用户画像更新

【故障排除】

问题1：Multi-Agent v2.0选项不显示
解决：
  1. 检查multi_agent.py是否在同一目录
  2. 确保导入无误：python -c "from multi_agent import multi_agent_app"
  3. 升级依赖：pip install langgraph --upgrade

问题2：切换版本后无法创建会话
解决：
  1. 必须在切换版本后重新创建会话
  2. 检查API配置（conf.py）
  3. 查看Web界面的错误消息

问题3：对话出现状态错误
解决：
  1. 确认选择的版本与会话版本一致
  2. 尝试新建会话
  3. 检查系统日志
