## 📌 案例1：中国平安 - 股息率数据质检

### 📥 用户输入
```
查询中国平安(000001.SZ)的股息率，并进行数据质量检查。

要求：
1. 获取最新的股息率数据
2. 检查该数据是否在合理范围内（通常0%-8%）
3. 如果异常，列出可能的原因
4. 给出使用建议（该数据能否信赖？是否需要修正？）
5. 提供替代验证方法
6. 必须明确说明"该数据是否可用于投资决策"
```

### 📤 系统输出

**数据来源与时间**
```
[DATE]: 2025-11-30
[SOURCE]: Tushare Pro API
[TIME_RANGE]: 20251128 至 20251128
```

**分析报告**

#### 1. 数据领域简介
股息率（Dividend Yield）是衡量上市公司现金分红水平的重要指标，计算公式为：每股年度现金分红 / 当前股价。该指标反映投资者通过分红获得的年化回报率，常用于评估蓝筹股或高分红股票的投资价值。根据市场经验，A股市场的股息率通常集中在0%-8%区间，超过8%可能提示估值异常或分红不可持续。

#### 2. 详细指标分析
- **最新股息率**：5.15%
- **数据状态**：可用
- **合理性判断**：在0%-8%的合理范围内
- **历史对比参考**：中国平安近年来股息率普遍维持在4.5%-6.0%之间，受盈利稳定性和分红政策支持，具备持续派息能力。当前5.15%处于其历史中枢偏上水平，但未出现显著偏离。

#### 3. 投资建议与使用评估

✅ **该数据可用于投资决策**

理由如下：
1. **数值合理**：5.15%位于典型安全区间内，符合保险行业龙头企业的分红特征
2. **来源可靠**：Tushare Pro API对接官方交易所及财报数据，具备权威性与时效性
3. **状态标记清晰**：系统返回"status: 可用"且注明"reason: 数据正常"，表明已完成基础逻辑校验
4. **无异常信号**：未发现极端值、缺失或类型错误等问题

🔹 **替代验证方法**：
- 查阅公司最新年报/公告中的每股分红金额和分红实施日股价，手动计算股息率
- 对比Wind、东方财富Choice等多平台数据，确认一致性
- 观察近12个月滚动股息率（TTM Yield），避免因单一时点波动造成误判

## 📌 案例2：中国神华 - 风险量化分析

### 📥 用户输入
```
对中国神华(601088.SH)进行风险分析，要求计算：

1. 最近252个交易日（约1年）的日线数据
2. 计算每日对数收益率：ln(Pt / Pt-1)
3. 计算年化波动率：日波动率 × √252
4. 计算最大回撤：从历史高点到低点的最大跌幅
5. 计算夏普比率：(年化收益 - 2%无风险率) / 年化波动率

最后输出：
- 完整的计算过程和公式
- 四个指标的数值结果
- 财务含义解释
- 数据准确性说明
```

### 📤 系统输出

**元数据定义**
```
[META]: 对数收益率定义 = log(当日收盘价/前一日收盘价)
[META]: 年化波动率 = 日收益率标准差 × √252
[META]: 最大回撤 = min((累计净值 - 滚动最高净值)/滚动最高净值)
[META]: 夏普比率 = (年化收益率 - 无风险利率) / 年化波动率
[META]: 无风险利率 = 2%
[META]: 实际数据行数 = 252 行
[DATE]: 2025-11-30
[SOURCE]: Tushare Pro API
[TIME_RANGE]: 2024-11-04 至 2025-11-28
```

**指标结果**

| 指标             | 数值 |
|------------------|------|
| 年化收益率       | 0.956% |
| 年化波动率       | 23.06% |
| 最大回撤         | -19.05% |
| 夏普比率         | -0.045 |

> 注：实际用于收益率计算的数据为 **251个交易日**，符合[META]中"样本量=251"的说明。

**财务含义解释**

- **年化波动率 23.10%**：表明该股票在过去一年的价格波动较为剧烈，高于市场平均水平（通常蓝筹股波动率在15%-20%），反映出一定市场敏感性或行业周期影响。

- **最大回撤 -19.05%**：意味着在过去一年中，投资者若在最高点买入，最多可能亏损近1/5本金。这一水平属于中等偏高风险，提示需关注下行保护策略。

- **夏普比率 -0.045**：小于零，说明其年化收益（0.956%）未能跑赢2%的无风险利率，承担了较高波动却未获得正向超额回报，投资性价比偏低。

- **整体评价**：尽管中国神华作为煤炭龙头具备稳定分红特性，但近期股价表现疲软，收益几乎持平且波动较大，导致风险调整后收益不佳。

## 📌 案例3：海康威视 - 财务健康度分析

### 📥 用户输入
```
查询海康威视(002415.SZ)，完整输出一份包含完整证据链的分析报告。

要求：
1. 获取该公司最近3年的主要财务指标
2. 计算"净现比" = 经营现金流 / 净利润
3. 计算"应收账款占比" = 应收账款 / 总营收
4. 基于这两个指标分析：该公司是否存在"赊销冲业绩"或"回款困难"？

关键要求：
- 每个财务指标都要明确标注 [DATE] 和 [SOURCE]
- 每个计算都要明确给出 [META] 定义（公式）
- 每个结论都要说明"基于的是哪个[DATA]数据"
- 如果无法获取某个数据，要说明 [WARNING] 和 [FALLBACK]方案
```

### 📤 系统输出

**数据来源与时间**
```
[DATE]: 2025-11-30
[SOURCE]: Tushare Pro API
```

**计算逻辑**
```
[META]: 净现比 = 经营活动产生的现金流量净额 / 归属于母公司股东的净利润
[META]: 应收账款占比 = 应收账款 / 总营业收入
```

**具体数据**
```
[DATA]: 2025年净现比 = 1.470（基于：经营现金流136.97亿元 / 净利润93.19亿元）
[DATA]: 2025年应收账款占比 = 1.975（基于：应收账款1298.63亿元 / 营收657.58亿元）
[DATA]: 2024年净现比 = 0.944（基于：经营现金流53.43亿元 / 净利润56.57亿元）
[DATA]: 2024年应收账款占比 = 2.975（基于：应收账款1244.15亿元 / 营收418.18亿元）
```

> 注：尽管请求为"最近3年"，但实际返回数据仅包含2024年与2025年两个年度，且部分条目重复。分析基于去重后的有效年度数据。

**分析结论**

基于上述[DATA]数据：

1. **2025年净现比为1.470**（[DATA]），高于1，表明当年净利润有较强的现金流支撑，盈利质量改善。

2. **2024年净现比仅为0.944**（[DATA]），低于1，说明当年约5.6%的净利润未能转化为现金流入，存在一定的"纸面利润"风险。

3. **2025年应收账款占比高达1.975倍营收**（[DATA]），即应收账款几乎是全年营收的两倍，远超合理水平。

4. **2024年应收账款占比更达2.975倍营收**（[DATA]），意味着每1元收入对应近3元的应收账款挂账，严重背离正常商业逻辑。

综上，结合[META]定义与[DATA]表现，海康威视在2024–2025年间存在明显的"**赊销冲业绩**"嫌疑。尤其2024年，在净利润未大幅增长的情况下，应收账款规模激增，而经营性现金流反而低于净利润，反映出**销售回款困难、资金占用严重**的问题。虽2025年净现比回升，但应收账款占比仍畸高，回款压力仍未根本缓解。

**风险提示**

根据[META]和[DATA]，存在以下风险：
- 高应收账款占比可能导致坏账计提增加，未来侵蚀利润
- 若客户集中于政府或国企项目，回款周期长将加剧营运资金压力
- 存在财务操纵嫌疑，需警惕"收入虚增+现金流滞后"的报表粉饰风险

- [WARNING] 原始数据中"无法找到应收账款字段"，系统[FALLBACK]使用代理变量可能导致偏差，建议通过财报附注核实"应收账款"确切数值以提升准确性。

## 💡 系统特色说明

### 1. 数据溯源完整性
- **每个数据都标注**：`[DATE]`（时间）、`[SOURCE]`（来源）、`[META]`（公式定义）
- **每个结论都引用**：明确说明"基于哪个[DATA]得出"
- **异常情况标记**：`[WARNING]`警告 + `[FALLBACK]`降级方案

### 2. 自我修复机制
- 代码报错自动重试（最多3次）
- 数据缺失自动寻找替代变量
- 异常值自动识别并告警（如股息率>10%）

### 3. 多层质检保障
- **Coder层**：强制输出元数据标签
- **Reviewer层**：数据验证+逻辑检查
- **ErrorHandler层**：三级错误分类处理
