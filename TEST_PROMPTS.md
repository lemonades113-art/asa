# 🧪 系统能力测试 Prompt 集合

## 📊 难度分级说明

- **L1 (基础)**: 单一任务，直接调用 API
- **L2 (简单)**: 两个步骤，需要基本的逻辑组合
- **L3 (中等)**: 三个步骤，涉及多个 Agent 协作
- **L4 (困难)**: 错误恢复，需要自动重规划
- **L5 (极限)**: 多轮交互，考验完整系统能力

---

**预期输出**: 10 行股票数据表

## 🟠 L3 - 中等复杂度

### Prompt 3.1: 完整的技术分析流程

```
对上证指数（000001.SH）进行技术分析：

1. 获取最近 60 个交易日的 OHLC 数据

3. 根据这些指标判断当前走势（上升/下降/震荡）
4. 生成一份专业的技术面分析报告
```

**测试内容**:
- ✅ 大量数据处理（Coder 计算复杂度高）
- ✅ 多指标计算（Numpy/Pandas）
- ✅ 技术分析逻辑（Coder 需要金融知识）
- ✅ Reviewer 专业报告生成
- ✅ 完整的 Supervisor + Coder + Reviewer 流程

**预期时间**: 5-10 秒  
**预期输出**: 
- 数据表格（60行）
- 三条线图（如果支持画图）
- 专业技术面分析报告（200+ 字）

**难点**: 
- 需要正确实现 SMA 和 RSI 算法
- Reviewer 需要给出有意义的分析

## 🟣 L4.5 - 困难-进阶（两轮完整工作流）

### Prompt 4.5: 高股息股票筛选器（专业版）

```
我是一个保守型投资者，想建立一个"高股息股票筛选器"来寻找稳健的分红股。
请按以下步骤帮我执行：

【第一轮】基础数据采集与准备
1. 从沪深 300 指数成分股中，筛选股息率最高的 5 支股票
2. 获取这 5 支股票最近 3 个月的收盘价数据
3. 计算每支股票的 3 个月价格涨幅（%）和波动率（日度收益率标准差）
4. 生成一份数据汇总，包括：股票代码、股息率、3个月涨幅、波动率

【第二轮】综合评分与分析
1. 基于股息率和波动率为每支股票打分（1-10分）
   - 股息率高 → 加分
   - 波动率低 → 加分（反映稳定性）
2. 生成评分排名表
3. 给出 Top 2 推荐股票，说明它们为什么适合保守型投资者：
   - 分红收益有多稳定
   - 价格波动有多小
   - 最适合的持仓期限
   - 预期年化收益

请确保最后的分析报告包含：专业的数据表格、逻辑清晰的评分依据、针对保守型投资者的具体建议。
```

**测试内容**:
- ✅ 两轮完整的任务流程（简化但保留核心）
- ✅ 多步骤编排（考验 Supervisor）
- ✅ 中等规模的数据处理和计算（考验 Coder）
- ✅ 金融知识和专业表达（考验 Reviewer）
- ✅ 错误恢复和降级方案（考验 ErrorHandler）
- ✅ 用户画像更新（ProfileUpdater）

**预期时间**: 15-25 秒  
**预期输出**: 
- 初始数据表（5×3 列）
- 评分表（5×3 列）
- 专业分析和建议（200+ 字）

**难点**:
- API 调用量较大（易触发限流，需降级）
- 需要多次数据聚合
- 需要综合性的金融评分逻辑
- 验证 Reviewer 的分析能力

### Prompt 5.2: 多轮优化和反馈

```
我是一个基金经理，想开发一个"自适应投资组合管理器"。

【第一轮】初始扫描
1. 基于之前的高股息筛选结果，列出最值得投资的 3 只个股
2. 给出具体的配置比例建议（按评分加权）
3. 预估的年收益率和风险等级

【第二轮】根据我的反馈优化
假设我告诉你："我比较看好科技股，但不想要金融股"
请重新调整建议。

【第三轮】再次优化
假设市场突然暴跌 5%，请更新你的风险评估和建议。

【第四轮】学习和记忆
根据这个对话，记住我的偏好（喜欢科技，不喜欢金融，风险承受力中等），
为后续建议提供个性化参考。
```

**测试内容**:
- ✅ 多轮对话记忆（Message History）
- ✅ 动态调整能力（Supervisor 重规划）
- ✅ 用户偏好学习（ProfileUpdater）
- ✅ 实时场景响应（市场变化）
- ✅ 个性化服务（基于用户画像）
- ✅ 完整的四轮交互流程

**预期时间**: 40-70 秒（含多轮交互）  
**预期输出**: 
- 第一轮：3 支股票建议表 + 配置 + 风险评分
- 第二轮：调整后的配置（应该减少金融，增加科技）
- 第三轮：应急调整方案
- 第四轮：用户画像学习确认

**难点**:
- 验证系统是否真的学习了用户偏好
- 验证多轮对话的连贯性和上下文记忆
- 验证动态调整的合理性和实时响应能力
