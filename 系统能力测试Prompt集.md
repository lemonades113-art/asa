# 🧪 AI金融分析系统 - 能力测试Prompt集

> **设计理念**：按照**5个真实金融场景**设计，从基础到复杂，展现系统的全面能力。

---

## 第一类：基础财务分析 (选股视角)

### P1.1 单只股票查询
```
请查询中国平安(000001.SZ)的最新数据：PE、PB、股息率。用表格输出。
```

### P1.2 多只股票对比
```
对比这3支银行股的PE、PB和股息率：
- 中国平安(000001.SZ)
- 招商银行(600036.SH)
- 工商银行(601398.SH)

给出排序和推荐。
```

### P1.3 选股配置系统
```
帮我从A股中筛选15支股票，满足：
- 股息率>4% 且 <8%
- PE倍数10-30倍
- 过去3年分红稳定增长
- 市值>100亿元
- 行业分散（金融不超30%）

给出配置权重和预期年收益。
```

## 第二类：技术面量化分析 (交易员视角)

### P2.1 MACD与布林带分析
```
分析中信证券(600030.SH)最近120个交易日的MACD和布林带：
1. 计算MACD（快线12、慢线26、信号线9）
2. 计算布林带（20日SMA ± 2倍标准差）
3. 找出金叉/死叉信号
4. 给出短期买卖建议
```

### P2.2 波动率与回撤分析
```
对中国神华(601088.SH)分析最近1年(252个交易日)的：
1. 年化波动率
2. 最大回撤
3. 夏普比率（无风险率2%）
4. 投资风险评分
```

### P2.3 异常异动扫描
```
请帮我扫描创业板指(399006.SZ)成分股中的"异常活跃股"：
1. 获取所有成分股今天的行情
2. 筛选出同时满足以下条件的股票：
   - 涨幅 > 5%
   - 换手率 > 10%
   - 量比 > 2.0
3. 输出"短线爆发潜力股"名单
4. 提示高位回调风险
```

## 第三类：投资组合风险管理 (基金经理视角)

### P3.1 相关性与对冲分析
```
我持仓50%贵州茅台 + 50%五粮液。
1. 计算这两只股票的相关系数
2. 从"电力"或"银行"板块推荐1-2只相关性<0.5的对冲股票
3. 计算加入对冲品种后的新组合波动率
4. 压力测试：大盘跌10%/20%，组合预期跌幅
```

### P3.2 完整的投资决策流程
```
我是保守型投资者，想建立"高分红+低风险"的组合。

请帮我：
1. 从沪深300中筛选股息率>4%的所有股票
2. 计算每支股票的年化波动率
3. 按"股息率60%+波动率稳定性40%"的权重评分
4. 推荐Top 5股票
5. 设计最优配置权重
6. 计算预期年收益和风险
7. 给出风险提示
```

## 第四类：宏观与行业轮动 (策略师视角)

### P4.1 进攻vs防守情绪判断
```
分析当前市场情绪：
1. 对比宁德时代(300750.SZ)与长江电力(600900.SH)
   最近3个月的涨幅
2. 计算相对强弱(科技涨幅-红利涨幅)
3. 判断资金是进攻还是防守
4. 给出配置建议
```

### P4.2 行业深度分析
```
分析房地产行业：
1. 列出市值前5的房企(万科、保利等)
2. 对比其PE、PB、股息率和3年利润增速
3. 行业估值与5年历史对比
4. 当前是高估还是低估
5. 给出配置建议
```

### P4.3 新能源产业链分析
```
请为"新能源汽车产业链"做完整分析：
1. 列出产业链主要企业(整车、电池、充电、材料共10+支)
2. 汇总它们的财务指标和估值水平
3. 分析产业链各环节的景气度
4. 对标同期涨跌幅，找最值得配置的环节
5. 给出配置建议(哪个环节投资比例最高，为什么)
6. 风险评估和后续监控指标
```

## 第五类：财务排雷检测 (审计视角)

### P5.1 海康威视财务排雷
```
请对海康威视(002415.SZ)进行财务健康度排雷：
1. 获取最近3年的"经营活动现金净流量"和"净利润"
2. 计算"净现比"（现金流/净利润），如果连续小于1，说明利润是纸面的
3. 获取最近3年的"应收账款"和"总营收"，计算占比
4. 综合判断是否存在"赊销冲业绩"或"回款困难"的财务风险
```

### P5.2 多维度估值评估
```
我要评估某科技公司IPO定价。
信息：2024年预计收益10亿，净利润1.5亿，增速45%。

请：
1. 从A股中找5-8家同类型科技公司(云计算/SaaS)
2. 对标它们的PE、PEG、PS
3. 计算同行业平均估值倍数
4. 基于对标给出合理估值范围
5. 评估50倍PE的定价是高估还是低估
```

### P5.3 系统自我修复展示
```
请分析贵州茅台(600519.SH)的股息率。
（这会触发系统发现之前的"515%股息率"问题并自动修复）
```

## 核心观察指标

运行每个Prompt时，观察：

```
✅ 【数据准确性】
  - 查询的数据是否与官方一致
  - 计算公式是否正确（PE、波动率等）

✅ 【逻辑完整性】
  - 是否回答了用户的所有问题
  - 结论是否有数据支撑

✅ 【专业性】
  - 报告格式是否专业
  - 风险提示是否充分
  - 建议是否可操作

✅ 【容错能力】
  - 是否能处理数据缺失（自动降级方案）
  - 是否能识别异常值（如515%股息率）
  - 是否有完整的错误重试机制

✅ 【响应速度】
  - 预期时间是否符合
  - 是否有缓冲（用了缓存机制）
```
