# 🎯 第一层（感知层）：五粮液 - 数据实时性

## 案例1：五粮液（000858.SZ）- BIAS 分析

### 背景：股价数据的"罗生门"

**场景**：同一时间窗口，向两个系统询问五粮液的当前股价与技术面

**核心问题**：通用模型是否能准确反映**当下的**市场价格？

---

### 📊 核心对比

| 指标 | 通用模型 | AI Agent | 真实行情(东方财富) | **关键发现** |
|------|------|--------|----------------|---------------|
| **收盘价** | **148.60元** | **114.96元** | **114.96元** | 💥 **偏差 +29.2%** |
| **MA60** | 142.35元（向上） | 120.92元（向下） | ~120元 ||
| **BIAS(60)** | +4.39%（多头） | -4.93%（弱势） | 弱势 | *|
| **技术结论** | 偏强/正常波动 | 偏弱/缺乏反弹 | 弱势整理 | ✅ |

---

### 🔍 原因分析

**通用模型的数据来源**：
- 使用的 **148.60元** 是五粮液在 **2024年11月附近的历史高点**
- 由于不具备实时数据接口，模型在数据截止期后使用了过期缓存
- 这不是"幻觉"，而是架构限制

**AI Agent 的数据获取**：
- 通过 Tushare Pro API 直接调用，实时获取最新收盘价 **114.96元**
- 与流行软件（东方财富）**100% 同步**
- 基于真实数据计算的 BIAS 值准确反映了当前市场情绪

---

### 💡 验证结论

**两个系统在本层的差异根源**：
- 通用模型：基于训练数据（滞后）+ 语义推断
- AI Agent：基于实时API（同步）+ 代码计算

**这不是优劣对比，而是定位差异**：
- 通用模型擅长**解释**（为什么五粮液涨/跌）
- AI Agent 擅长**观察**（现在五粮液在哪里）

# 🧠 第二层（认知层）：茅台+紫金 - 计算透明性与风险识别

## 案例2：贵州茅台（600519.SH）- PE-TTM 估值

**用户需求**：分析贵州茅台当前PE-TTM处于历史什么水位？

**Supervisor 任务分解**：
```

✅ 获取5年历史PE-TTM数据
✅ 计算5年历史PE均值
✅ 计算当前PE的历史分位点
✅ 估值判断（±20%阈值）
✅ 输出分析报告
```

---

### 📊 核心对比数据

| 指标 | AI Agent | 通用模型 | 差异分析 |
|------|--------|--------|--------|
| **当前 PE-TTM** | **19.89** | **19.89** | ✅ **完全一致** |
| **5年平均 PE** | **35.26** | **25.52** | ⚠️ 算法差异 |
| **历史分位点** | **1.32%** | **1.13%** | 轻微差异 |
| **结论** | 明显低估 | 明显低估 | ✅ **一致** |

---

### 🔍 **优势对比：计算透明性**

**AI Agent 的做法**：
```python
# 1. 直接调用 API 获取数据
df_hist = pro.daily_basic(ts_code='600519.SH', start_date='20201207')

# 2. 计算分位点（数学公式，完全透明）
percentile = rank / len(samples)  # 当前PE排名 / 总样本数

# 3. 完整的元数据说明
print("[META]: 分位点计算方法 = 当前PE排名 / 总样本数")
print("[DATA]: 历史分位点 = 1.32%")
```

**通用模型的做法**：
```
"过去5年PE平均值：约25.52（根据多个数据源综合）"
→ 用户不知道这个数字怎么来的
```

**差异本质**：
- AI Agent：**白盒**，每一步可追溯
- 通用模型：**黑盒**，基于训练数据合成

---

### 🔍 **优势对比：风险识别**

**AI Agent Reviewer 报告中的深度观察**：

**这是 System 2 思维**：不仅看数字，还看背后的假设陷阱。

### 🔴 **AI Agent 的审慎反应**
