# 🟢 新增Level 2.5: 技术面量化分析

## 场景 2.5.1 - MACD与布林带的技术面体检

**难度**：⭐⭐  
**预期耗时**：10-15秒  
**成功率**：80%-90%  
**测试内容**：指标计算、信号识别、量价分析

### Prompt

```
请对中信证券(600030.SH)进行深度技术面体检：

【第一步】数据获取
1. 获取最近120个交易日的日线数据(开盘价、收盘价、成交量)

【第二步】指标计算
2. 计算MACD指标：
   - 快线(EMA12) = 12周期指数移动平均
   - 慢线(EMA26) = 26周期指数移动平均
   - MACD线 = EMA12 - EMA26
   - 信号线(DEM) = MACD的9周期EMA
   - 柱状图(MACD柱) = MACD - DEM

3. 计算布林带(Bollinger Bands)：
   - 中轨 = 20周期简单移动平均(SMA20)
   - 上轨 = 中轨 + 2×标准差
   - 下轨 = 中轨 - 2×标准差

【第三步】信号识别
4. 判断当前的技术形态：
   - 当前收盘价处于布林带的什么位置？(上轨/中轨/下轨)
   - MACD是否出现金叉信号(快线从下穿上)？
   - MACD是否出现死叉信号(快线从上穿下)？
   - MACD柱状图是否转正(看涨信号)？

【第四步】量价分析
5. 分析近5日的成交量变化：
   - 最近5日平均成交量与过去20日平均比，是否异常放量？
   - 价格创新高时，成交量是否同步创新高(量价齐升，看涨信号)？
   - 价格创新低时，成交量是否也创新低(量价齐跌，看跌信号)？

【第五步】综合判断
6. 基于以上指标，给出短期操作建议：
   - 当前是"买入信号"、"卖出信号"、还是"观望"？
   - 如果是买入信号，目标位置和止损位置是什么？

输出：
- 一份包含MACD线图、布林带走势图的技术分析报告
- 清晰的买卖点位建议
- 风险提示(如高位追入的风险、低位补仓的机会)
```

### 评估标准

- ✅ MACD计算是否准确（快线、慢线、DEM的算法）
- ✅ 布林带的上中下轨是否正确（SMA20和2倍标准差）
- ✅ 金叉/死叉的判断逻辑是否清晰
- ✅ 量价背离的识别是否正确
- ✅ 最终建议是否有根据、有风险提示

### 核心难点

```
【难点1】EMA计算
EMA = 当期收盘价 × α + 上期EMA × (1-α)
其中 α = 2 / (n+1)，n是周期数

对Coder的挑战：需要正确实现递推算法

【难点2】金叉判断
需要识别"MACD线从下穿上"这个转折点
不能简单地看"MACD > 0"（这样的股票太多了）

对Coder的挑战：需要逐日比较，找出穿越点

【难点3】Reviewer的专业性
不能只说"MACD金叉是看涨信号"
要说"XX日MACD出现金叉，结合布林带中轨支撑和量价齐升，
      形成标准的看涨形态，短期有望向上突破至XX位置"

对Reviewer的挑战：需要理解技术面的逻辑
```

# 🟠 新增Level 3.5: 投资组合风险管理

## 场景 3.5.1 - 组合相关性与风险对冲

**难度**：⭐⭐⭐  
**预期耗时**：15-25秒  
**成功率**：70%-80%  
**测试内容**：多资产相关性计算、VaR测试、对冲建议

### Prompt

```
我当前的投资组合存在风险集中问题，请帮我进行深度的风险诊断与对冲方案：

【当前持仓】
- 50% 贵州茅台(600519.SH)  [消费龙头]
- 50% 五粮液(000858.SZ)    [消费龙头]
总规模：1000万元

【您的风险担忧】
"这两个都是消费股，我担心一旦消费板块下跌，会'火烧连营'"

【分析要求】

【第一步】相关性诊断
1. 获取这两只股票最近1年(252个交易日)的日线数据

2. 计算日收益率：
   R_t = ln(P_t / P_{t-1})  [对数收益率，更符合金融学原理]

3. 计算皮尔逊相关系数(Pearson Correlation Coefficient)：
   ρ = Cov(R_茅台, R_五粮液) / (σ_茅台 × σ_五粮液)
   
   ├─ 如果ρ接近1：高度正相关，一起涨跌（风险集中）
   ├─ 如果ρ接近0：无相关性（可以分散风险）
   └─ 如果ρ<0：负相关（可以互相对冲）

4. 计算组合的整体波动率：
   σ_组合 = √[w₁²σ₁² + w₂²σ₂² + 2×w₁×w₂×ρ×σ₁×σ₂]
   其中w₁=50%, w₂=50%

【第二步】风险对冲建议
5. 从"电力"或"银行"板块中，推荐1-3只与茅台相关性低(ρ<0.5)或负相关(ρ<0)的股票
   
   搜索逻辑：
   ├─ 电力股代表：长江电力(600900.SH)、华能国际(600011.SH)
   ├─ 银行股代表：工商银行(601398.SH)、招商银行(600036.SH)
   └─ 计算它们与"茅台"的相关系数，找相关性最低的

6. 推荐的对冲方案：
   假设你用10%的新钱来加仓对冲品种，推荐的新配置是：
   - 40% 贵州茅台
   - 40% 五粮液
   - 20% [对冲品种] (选最相关性最低的)

【第三步】风险对比
7. 计算对冲前后的组合风险变化：
   - 原组合波动率：σ_原
   - 新组合波动率：σ_新
   - 风险下降幅度：(σ_原 - σ_新) / σ_原 × 100%

8. 极端情景压力测试：
   - 假设消费板块暴跌10%，你的原组合会跌多少？
   - 假设对冲后，整个组合会跌多少？
   - 风险是否得到有效降低？

【第四步】综合建议
9. 给出最终的"风险管理方案"，包括：
   - 当前组合的风险水平评级(低/中/高)
   - 为什么这三只股票的组合比两只更优(多样化收益)
   - 后续的监控指标(相关性、波动率、最大回撤)

输出：
- 相关系数矩阵(3×3)
- 风险对比表(原配置 vs 对冲配置)
- 压力测试结果表
- 风险管理建议书(500字+)
```

### 评估标准

- ✅ 相关系数计算是否准确
- ✅ 对冲品种的筛选逻辑是否正确
- ✅ 新组合波动率的计算公式是否正确
- ✅ 压力测试是否基于实际的贝塔值
- ✅ 最终建议是否可操作(具体说买多少、卖多少)

### 核心难点

```
【难点1】相关系数矩阵计算
需要用NumPy/Pandas的corr()函数或手写公式
对Coder的挑战：确保矩阵对称、数值正确

【难点2】优化对冲品种的选择
不能随意选，要找与茅台相关性最低的
需要批量计算多只股票与茅台的相关系数，排序，取Top 1

对Coder的挑战：跨资产的相关性计算和排序

【难点3】Reviewer理解风险管理逻辑
不能只说"加一只负相关的股票"
要说"贵州茅台(消费龙头)与长江电力(电力龙头)的相关系数为-0.15，
      说明两者走势基本独立。加入长江电力后，组合的波动率从18.5%
      降低到16.2%，风险下降12.3%，同时保持了年化收益率。"

对Reviewer的挑战：需要理解现代投资组合理论(MPT)
```

# 🔴 新增Level 2.6: 行业轮动与情绪分析

## 场景 2.6.1 - 进攻 vs 防守的资金流向判断

**难度**：⭐⭐  
**预期耗时**：8-12秒  
**成功率**：85%-95%  
**测试内容**：行业龙头对比、相对强弱判断、风格判断

### Prompt

```
请帮我分析当前市场的"进攻"与"防守"资金流向：

【第一步】龙头对比
1. 获取最近3个月(约60个交易日)的涨跌数据：
   
   进攻方(科技/消费)：
   - 立讯精密(002475.SZ)   [消费电子龙头]
   - 沪深300科技权重 或 宁德时代(300750.SZ) [新能源龙头]
   
   防守方(红利/公用事业)：
   - 长江电力(600900.SH)   [电力龙头]
   - 中国平安(000001.SZ)   [金融龙头]

2. 计算每支龙头的3个月涨幅（%）

【第二步】相对强弱分析
3. 计算"进攻方相对强弱"：
   相对收益 = (进攻方涨幅) - (防守方涨幅)
   
   ├─ 如果相对收益 > 5%：资金流向科技/成长 → 市场情绪"进攻"
   ├─ 如果相对收益 < -5%：资金流向电力/银行 → 市场情绪"防守"
   └─ 如果 -5% < 相对收益 < 5%：拉锯阶段，观望

4. 绘制"进攻/防守的强弱指标"走势图：
   Y轴 = 相对收益
   X轴 = 时间(过去3个月，按周统计)
   
   用这条曲线看市场情绪的转变：
   - 曲线向上 → 情绪改善，资金进攻
   - 曲线向下 → 情绪恶化，资金防守

【第三步】资金流向推断
5. 基于涨幅，推断"机构资金"最近在做什么：
   
   场景A：立讯精密涨15%，长江电力涨2%
   → 推断：机构在加码科技股，对经济前景乐观
   
   场景B：立讯精密涨2%，长江电力涨12%
   → 推断：机构在加仓防守品种，对经济前景悲观，规避风险

【第四步】风格建议
6. 给出当前的"最优仓位风格"建议：
   
   如果进攻方强势：
   ├─ 建议：加大科技/成长股配置，减少电力/金融配置
   ├─ 理由：资金在流向这个方向，跟随资金
   └─ 风险：高估值，容易回调，需设好止损
   
   如果防守方强势：
   ├─ 建议：加大电力/金融配置，减少科技/成长配置
   ├─ 理由：资金在避险，意味着风险事件可能发生
   └─ 风险：错过后续行情反弹，需设好止盈

输出：
- 近3个月的"进攻 vs 防守"相对收益曲线图
- 当前的市场情绪判断表
- 仓位风格建议(激进/均衡/保守)
```

### 评估标准

- ✅ 龙头股涨幅计算正确
- ✅ 相对强弱的判断逻辑清晰
- ✅ 资金流向推断有依据

# 🟡 新增Level 1.6: 异常异动扫描

## 场景 1.6.1 - 创业板异常活跃股扫描

**难度**：⭐  
**预期耗时**：5-10秒  
**成功率**：90%+  
**测试内容**：批量筛选、条件过滤、异常值识别

### Prompt

```
请帮我扫描创业板(399006.SZ)中的"异常活跃股"，找出可能的短线机会：

【第一步】获取成分股数据
1. 获取创业板指数的所有成分股列表(大约500-800支)
   提示：这些股票的市场代码(market字段)为'创业板'

2. 获取这些股票"今天"的行情数据(日期为最新交易日)，包括：
   - 收盘价
   - 涨跌幅 (%)
   - 成交量 (股数)
   - 成交额 (元)
   - 换手率 (%)

【第二步】筛选异常活跃的股票
3. 应用以下筛选条件（同时满足）：
   
   条件A：价格动能强
   └─ 涨幅 > 5% (说明有主力在买)
   
   条件B：成交极度活跃
   └─ 换手率 > 10% (说明有大量交易)
   
   条件C：量比异常
   └─ 量比 > 2.0 (相对平均成交量，今天的量是平时的2倍以上)
   
   【这三个条件同时满足的股票，就是"异常活跃股"】

【第三步】进一步分类
4. 将筛选出的股票进行分类：
   
   "强势突破"(涨幅>10% AND 换手>15%)：
   - 可能原因：重大利好、资金主动建仓
   - 风险：高位追入容易被套
   
   "温和放量"(涨幅5-10% AND 换手10-15%)：
   - 可能原因：中长期资金进场
   - 风险：相对较低
   
   "量价分离"(涨幅<5% BUT 换手>10%)：
   - 可能原因：主力洗盘、机构换手
   - 风险：走势不明确

【第四步】风险提示
5. 对这份名单进行"风险排雷"：
   - 是否有ST标签(暂停上市风险)？
   - 股价是否在历史高位(容易高位套牢)？
   - 最近30天的最高价是多少，当前是否已接近(涨幅还有空间吗)？

【第五步】输出清单
6. 生成"短线爆发潜力股"的排名表：
   
   排名 | 股票代码 | 股票名称 | 涨幅  | 换手率 | 量比  | 分类      | 风险等级
   ─────────────────────────────────────────────────────
   1   | 00xxxx  | xxxx   | +8.5% | 12%   | 2.3  | 强势突破  | 高
   2   | 00xxxx  | xxxx   | +6.2% | 11%   | 2.1  | 温和放量  | 中
   ...

输出：
- "异常活跃股"完整清单(按涨幅排序)
- 风险提示：为什么这些股票可能存在"高位回调风险"
- 操作建议：如果要短线追入，止损位置应该怎么设？
```

- ✅ 三个筛选条件是否正确应用(AND关系)

# 🔵 新增Level 2.7: 财务排雷检测

## 场景 2.7.1 - 海康威视的财务健康度排雷

**难度**：⭐⭐  
**预期耗时**：12-18秒  
**成功率**：75%-85%  
**测试内容**：跨表查询、财务比率计算、异常识别

### Prompt

```
请对海康威视(002415.SZ)进行"财务健康度排雷"，判断是否存在财务风险：

【第一步】获取财务数据
1. 从以下三个表获取最近3年(2022年、2023年、2024年)的数据：
   
   【利润表数据】- income表
   - 营业收入 (revenue)
   - 净利润 (net_profit)
   
   【现金流量表数据】- cashflow表
   - 经营活动产生的现金流净额 (operate_cash_flow)
   - 投资活动产生的现金流净额 (invest_cash_flow)
   
   【资产负债表数据】- balance表
   - 应收账款 (accounts_receivable)
   - 预付款项 (prepaid_exp)
   - 存货 (inventory)

【第二步】"净现比"分析 ⭐ 核心指标
2. 计算"净现比"(Net Profit to Operating Cash Flow Ratio)：
   
   净现比 = 经营现金流 / 净利润
   
   这个指标的含义：
   ├─ 如果净现比 = 1.0：完美，利润都转化成现金了 ✅
   ├─ 如果净现比 > 1.0：很好，现金流比利润还多(自由现金流充足)
   ├─ 如果净现比 0.5-0.9：一般，有一定的应收未收
   ├─ 如果净现比 < 0.5：危险信号❌，"利润是假的，钱没收到"
   │  原因可能是：
   │  ├─ 赊销冲业绩（先记收入，后收钱）
   │  ├─ 应收账款无法回收（坏账风险）
   │  └─ 存货积压（产品卖不出去）
   └─ 如果净现比 < 0：严重问题，经营现金流为负(公司在烧钱)

3. 计算最近3年的净现比：
   
   2022年：净现比 = ?
   2023年：净现比 = ?
   2024年：净现比 = ?
   
   ⚠️ 重点检查：是否连续下降？是否有一年跌破0.5？

【第三步】"应收账款占比"分析
4. 计算"应收账款占收入比"：
   
   应收占比 = 应收账款 / 营业收入
   
   这个指标的含义：
   ├─ 如果应收占比 < 10%：很好，销售回款及时
   ├─ 如果应收占比 10-30%：正常，这是销售过程中的自然应收
   ├─ 如果应收占比 30-50%：有点高，要监控
   ├─ 如果应收占比 > 50%：非常危险❌，说明：
   │  ├─ 产品严重滞销（没人要，只好记账）
   │  ├─ 赊销冲业绩（为了做大收入）
   │  └─ 客户信用下降（回款困难）
   └─ 如果应收占比急剧上升：特别警惕，可能是"造假信号"

5. 计算最近3年的应收占比：
   
   2022年：应收占比 = ?%
   2023年：应收占比 = ?%
   2024年：应收占比 = ?%
   
   ⚠️ 重点检查：是否逐年上升？是否偏离行业平均？

【第四步】"存货周转"分析
6. 计算"存货与收入比"(存货积压指数)：
   
   存货占比 = 存货 / 营业收入
   
   这个指标的含义：
   ├─ 如果存货占比 < 20%：正常
   ├─ 如果存货占比 20-40%：注意，可能有滞销风险
   ├─ 如果存货占比 > 40%：警惕，产品可能堆积
   
   【衡量标准】：这个指标应该相对稳定，不应该急剧上升

【第五步】综合"财务健康度"评分
7. 基于以上三个指标，对公司的财务健康度进行"信用评级"：
   
   ✅ A级(财务健康)：
   └─ 净现比 > 0.8 AND 应收占比 < 20% AND 存货占比 < 30%
   
   ⚠️ B级(轻度风险)：
   └─ 净现比 0.5-0.8 OR 应收占比 20-40% OR 存货占比 30-50%
   
   ❌ C级(中度风险)：
   └─ 净现比 0.2-0.5 OR 应收占比 40-60% OR 存货占比 50-70%
   
   🔴 D级(严重风险)：
   └─ 净现比 < 0.2 OR 应收占比 > 60% OR 存货占比 > 70%

【第六步】风险解读
8. 针对海康威视的具体情况，给出：
   
   如果存在风险征兆：
   ├─ 问题根源：这个问题是什么引起的？
   │  ├─ 行业景气度下降(大环境坏了)
   │  ├─ 产品竞争力下降(被超越了)
   │  ├─ 应收管理不善(账期太长)
   │  └─ 销售手段激进(为了冲数字)
   │
   ├─ 问题严重程度：是短期波动还是长期恶化？
   │
   └─ 投资建议：这个风险意味着什么？
      ├─ 价值被高估(PE可能不便宜了)
      ├─ 存在踩雷风险(应该回避或减仓)
      └─ 需要持续监控(等财务指标改善)

输出：
- 财务健康度评分表(3年数据对比)
- 风险诊断报告(为什么存在这些问题)
- 排雷建议(该不该买，或者该不该继续持仓)
```

### 评估标准

- ✅ 数据是否从正确的表中获取(income/cashflow/balance)
- ✅ 净现比、应收占比、存货占比的计算是否准确
- ✅ 3年数据对比是否完整(能看出趋势)
- ✅ 风险识别是否准确(不要过度解读)
- ✅ 最终建议是否合理(既不过度保守，也不过度乐观)
